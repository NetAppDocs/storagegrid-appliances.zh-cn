= 安装到机柜或机架中(SG6000)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
对于SG6060和SGF6024、您可以在机柜或机架中安装导轨、然后将控制器架、所有扩展架和计算控制器滑入导轨上。对于SG6060、请勿在每个磁盘架中安装驱动器、除非安装了这些磁盘架。

 a| 
SG6060

 a|
